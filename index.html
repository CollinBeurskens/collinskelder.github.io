FilterKeysSetter

FilterKeysSetter is een applicatie dat ervoor zorgt dat je keyboard sneller reageert.

Om FilterKeysSetter te openen moet je het .zip bestand uitpakken. Klik op "Download" open de downloads folder, zoek het FilterKeysSetter zip.

Windows 10 : klik met het rechter knopje op je muis op het zip bestand. Klik daarna op uitpakken, en dan op OK, wacht dan totdat het bestand is gedownload, zodra het bestand is gedownload open de folder, en dan op FilterKeysSetter.exe

Windows 11 : Klik met het rechter knopje op je muis op het zip bestand. Klik daarna op meer weergeven, daarna op uitpakken, en dan op OK, wacht dan totdat het bestand is gedownload, zodra het bestand is gedownload open de folder, en dan op FilterKeysSetter.exe

Geforce Experience

Geforce Experience is een applicatie dat ervoor zorgt dat je kills automatisch worden opgeslagen, het zorgt er ook voor dat je je driver's kunt updaten. Zodra je je driver's update krijg je (waarschijnlijk) meer FPS

Launcher Settings

Open Epic Games (NIET OP JE INTERNET BROWSER), klik rechtsboven op je profiel, klik dan op settings, scroll daarna helemaal naar beneden, totdat je Fortnite ziet, klik dan op Fortnite, dan op "Additional Command Line Arguments", en dan type daaronder in "-LANPLAY -NOSPLASH -USEALLAVAILABLECORES"

Prestaties aanzetten op windows

Klik op windows knop + R, type in "sysdm.cpl", klik vervolgens op geavanceerd, klik dan op je keyboard op TAB, en dan Enter, dan op Beste Prestaties, dan toepassen.
